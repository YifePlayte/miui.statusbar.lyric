# MIUI状态栏歌词

### 自动获取音乐软件当前播放的歌曲，并实时将歌词显示在状态栏

__支持MIUI12以后的MIUI__

#### 支持的音乐软件版本

- 酷狗音乐:v10.8.4 (需打开蓝牙歌词)
- 酷我音乐:v9.4.6.2 (需打开蓝牙歌词)
- 网易云音乐:v8.6.0 (完美适配)
- Aplayer:v1.5.7.9 (完美适配)
- QQ音乐:v10.17.0.11 (需打开蓝牙歌词和戴耳机)
